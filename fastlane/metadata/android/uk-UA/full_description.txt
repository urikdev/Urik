Клавіатура з пріоритетом конфіденційності для Android з обробкою на пристрої та зашифрованим сховищем.

**Введення**
- Введення свайпом та дотиком з адаптивним розпізнаванням жестів
- Довге натискання для варіацій символів
- Свайп по пробілу для керування курсором
- Альтернативні розкладки клавіатури (Dvorak, Colemak, Workman)
- Визначення апаратної клавіатури (автоматично ховається при підключенні)

**Інтелект**
- Перевірка орфографії на пристрої з використанням алгоритму SymSpell
- Розумна автокорекція (пропускає URL/електронну пошту, обробляє контекст пунктуації)
- Вивчення слів із зашифрованою базою даних SQLCipher
- Багатомовна підтримка, одночасна перевірка орфографії
- Пошук емодзі з підтримкою ключових слів

**Інтеграція**
- Підтримка вбудованого автозаповнення для менеджерів паролів (Android 11+)
- Історія буфера обміну із зашифрованим сховищем
- Динамічні теми Material You (Android 12+)

**Налаштування**
- Безліч тем (світлі, темні, користувацькі кольорові схеми)
- Налаштовувані розміри клавіш, тактильний зворотний зв'язок, поведінка при наборі
- Функції доступності (контраст WCAG AA, підтримка TalkBack)

**Конфіденційність**
- Жодної телеметрії, аналітики чи мережевих дозволів
- Вся обробка відбувається на пристрої
- Зашифроване локальне сховище для даних користувача
