Клавиатура с приоритетом конфиденциальности для Android с обработкой на устройстве и зашифрованным хранилищем.

**Ввод**
- Ввод свайпом и касанием с адаптивным распознаванием жестов
- Долгое нажатие для вариаций символов
- Свайп по пробелу для управления курсором
- Альтернативные раскладки клавиатуры (Dvorak, Colemak, Workman)
- Определение аппаратной клавиатуры (автоматически скрывается при подключении)

**Интеллект**
- Проверка орфографии на устройстве с использованием алгоритма SymSpell
- Умная автокоррекция (пропускает URL/электронную почту, обрабатывает контекст пунктуации)
- Изучение слов с зашифрованной базой данных SQLCipher
- Многоязычная поддержка, одновременная проверка орфографии
- Поиск эмодзи с поддержкой ключевых слов

**Интеграция**
- Поддержка встроенного автозаполнения для менеджеров паролей (Android 11+)
- История буфера обмена с зашифрованным хранилищем
- Динамические темы Material You (Android 12+)

**Настройка**
- Множество тем (светлые, тёмные, пользовательские цветовые схемы)
- Настраиваемые размеры клавиш, тактильная обратная связь, поведение при наборе
- Функции доступности (контраст WCAG AA, поддержка TalkBack)

**Конфиденциальность**
- Никакой телеметрии, аналитики или сетевых разрешений
- Вся обработка происходит на устройстве
- Зашифрованное локальное хранилище для данных пользователя
