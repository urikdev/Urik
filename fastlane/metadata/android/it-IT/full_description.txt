Tastiera orientata alla privacy per Android con elaborazione sul dispositivo e archiviazione crittografata.

**Input**
- Digitazione scorrevole e tocco con rilevamento adattivo dei gesti
- Pressione prolungata per variazioni di caratteri
- Scorrimento sulla barra spaziatrice per il controllo del cursore
- Layout di tastiera alternativi (Dvorak, Colemak, Workman)
- Rilevamento tastiera hardware (si nasconde automaticamente quando connessa)

**Intelligenza**
- Controllo ortografico sul dispositivo utilizzando l'algoritmo SymSpell
- Correzione automatica intelligente (salta URL/email, gestisce il contesto della punteggiatura)
- Apprendimento delle parole con database SQLCipher crittografato
- Supporto multilingue, controllo ortografico simultaneo
- Ricerca emoji con supporto per parole chiave

**Integrazione**
- Supporto per riempimento automatico in linea per gestori di password (Android 11+)
- Cronologia degli appunti con archiviazione crittografata
- Temi dinamici Material You (Android 12+)

**Personalizzazione**
- Temi multipli (chiaro, scuro, schemi di colori personalizzati)
- Dimensioni dei tasti configurabili, feedback aptico, comportamento di digitazione
- Funzionalità di accessibilità (contrasto WCAG AA, supporto TalkBack)

**Privacy**
- Nessuna telemetria, analisi o autorizzazioni di rete
- Tutta l'elaborazione avviene sul dispositivo
- Archiviazione locale crittografata per i dati dell'utente
