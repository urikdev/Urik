Клавиатура для Android с защитой конфиденциальности, свайп-набором, пользовательскими раскладками и поддержкой менеджеров паролей. Без отслеживания, 100% на устройстве и полностью с открытым исходным кодом.

## Возможности

**Ввод**
- Свайп-набор с геометрическим сопоставлением путей и обнаружением вершин
- Режимы одной рукой и разделённый с преобразованием координат
- Адаптивная раскладка для планшетов и складных устройств
- Долгое нажатие для вариантов символов и пользовательского назначения клавиш
- Долгое нажатие клавиши запятой для переключения клавиатуры (выбор IME)
- Свайп по пробелу для управления курсором
- Свайп по Backspace для удаления предыдущего слова
- Дополнительный ряд цифр
- Раскладки QWERTZ, AZERTY, Dvorak, Colemak, Workman, Bds и Hcesar
- Определение аппаратной клавиатуры

**Интеллект**
- Локальная биграммная модель для предсказания следующего слова
- Проверка орфографии на устройстве с использованием URIK (Ultra-compressed Ranked Input Korpus) — пользовательский двоичный формат словаря с обходом автомата Левенштейна
- Автокоррекция с настраиваемой паузой при ошибках; пропускает URL, email и пунктуацию
- Изучение слов с зашифрованной базой данных SQLCipher
- Отслеживание частоты слов конкретного пользователя
- 19 поддерживаемых языков с отдельной кнопкой переключения языка
- Режим объединённых словарей для многоязычного набора
- Поиск emoji с поддержкой ключевых слов

**Интеграция**
- Встроенная поддержка автозаполнения для менеджеров паролей (Android 11+)
- История буфера обмена с зашифрованным хранилищем

**Настройка**
- Более 15 встроенных тем с избранными и динамическими темами Material You (Android 12+)
- Пользовательские символы при долгом нажатии и назначение клавиш
- Ползунок интенсивности тактильного отклика
- Переключение автоматического регистра
- Настраиваемые размер клавиш, размер подписей, размер пробела и скорость курсора
- Управление выученными словами: просмотр, поиск и удаление записей
- Поддержка контрастности WCAG AA и TalkBack

**Конфиденциальность**
- Без телеметрии, аналитики или сетевых разрешений
- Локальная обработка на устройстве
- Зашифрованное локальное хранилище для данных пользователя (AES-256 + Android Keystore)
- Биометрическая аутентификация для доступа к выученным словам
- Экспорт и импорт словаря для резервного копирования или миграции
- Биграммные предсказания и частоты слов остаются только локальными (не экспортируются)

## Конфиденциальность

Вся обработка происходит на устройстве. Никакие данные не покидают ваш телефон. Пользовательский словарь и выученные слова хранятся в зашифрованной локальной базе данных.

Мы отвечаем нашим пользователям, а не инвесторам.
