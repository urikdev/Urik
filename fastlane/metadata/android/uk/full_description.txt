Клавіатура для Android з акцентом на конфіденційність, свайп-набором, користувацькими розкладками та підтримкою менеджерів паролів. Без відстеження, 100% на пристрої та повністю з відкритим кодом.

## Можливості

**Введення**
- Свайп-набір з геометричним зіставленням шляхів та виявленням вершин
- Режими однієї руки та розділений з перетворенням координат
- Адаптивна розкладка для планшетів та складних пристроїв
- Довге натискання для варіантів символів та користувацького призначення клавіш
- Довге натискання клавіші коми для перемикання клавіатури (вибір IME)
- Свайп по пробілу для керування курсором
- Свайп по Backspace для видалення попереднього слова
- Необов'язковий ряд цифр
- Розкладки QWERTZ, AZERTY, Dvorak, Colemak, Workman, Bds та Hcesar
- Визначення апаратної клавіатури

**Інтелект**
- Локальна біграмна модель для передбачення наступного слова
- Перевірка орфографії на пристрої з використанням URIK (Ultra-compressed Ranked Input Korpus) — користувацький двійковий формат словника з проходженням автомата Левенштейна
- Автокорекція з налаштовуваною паузою при помилках; пропускає URL, email та пунктуацію
- Вивчення слів із зашифрованою базою даних SQLCipher
- Відстеження частоти слів конкретного користувача
- 19 підтримуваних мов із спеціальною кнопкою перемикання мови
- Режим об'єднаних словників для багатомовного набору
- Пошук emoji з підтримкою ключових слів

**Інтеграція**
- Вбудована підтримка автозаповнення для менеджерів паролів (Android 11+)
- Історія буфера обміну із зашифрованим сховищем

**Налаштування**
- Понад 15 вбудованих тем із вибраними та динамічними темами Material You (Android 12+)
- Користувацькі символи при довгому натисканні та призначення клавіш
- Повзунок інтенсивності тактильного зворотного зв'язку
- Перемикач автоматичного регістру
- Налаштовувані розміри клавіш, підписів, пробілу та швидкість курсора
- Керування вивченими словами: перегляд, пошук та видалення записів
- Підтримка контрастності WCAG AA та TalkBack

**Конфіденційність**
- Без телеметрії, аналітики або мережевих дозволів
- Локальна обробка на пристрої
- Зашифроване локальне сховище для даних користувача (AES-256 + Android Keystore)
- Біометрична автентифікація для доступу до вивчених слів
- Експорт та імпорт словника для резервного копіювання або міграції
- Біграмні передбачення та частоти слів залишаються лише локальними (не експортуються)

## Конфіденційність

Вся обробка відбувається на пристрої. Жодні дані не залишають ваш телефон. Словник користувача та вивчені слова зберігаються у зашифрованій локальній базі даних.

Ми відповідаємо нашим користувачам, а не інвесторам.
